Programming Languages

Term Definition
Machine Language Sequences of instructions that are encoded as numeric digits
Debugging Process of locating and correcting errors in program
Assembler Convert mnemonic expressions into machine language instructions
Assembly Language A mnemonic system for representing programs is collectively
Translator Translates programs expressed in high-level primitives into machine-language programs
Compiler Compiles several machine instructions into short sequences to simulate the activity requested by a single high-level primitive
Interpreter Similar to translators except that they executed the instructions as they were translated instead of recording the translated version for future use
Cross-Platform Software Software that is independent of an operating system’s design as well as the machine’s hardware design and is therefore executable throughout a network.
Natrual Language Languages such as English, German, and Latin
Formal Language Programming languages
Machine Independence Program written in a third-generation language could theoretically be used on any machine simply by applying the appropriate compiler.
High-Level Programming Language Make computer programs both easier for human authors to write, and easier for subsequent readers to understand.
Object-Oriented Programming (OOP) A software system is viewed as a collection of units, called objects, each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects. Together, these objects interact to solve the problem at hand.
Scripting Language A subset of the imperative programming languages is the collection of languages
Data Types
  • Integer: numeric data consisting of whole numbers
  • Float: numeric data that might contain values other than whole numbers
  • String: Text is normally viewed as a long string of characters
Primitive Data Type The data types that are included as primitives in a programming language, such as "int" for integer and "char" for character
Data Structure The conceptual shape or arrangement of data
Python Collection Data Structures
  • List: List items are ordered, changeable, and allow duplicate values.
  • Tuple: A tuple is a collection which is ordered and unchangeable
  • Set: A set is a collection which is unordered, unchangeable, and unindexed
  • Dictionary:A dictionary is a collection which is ordered, changeable and do not allow duplicates.
Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
Local Variable A variable declared within a function
Global Variable Variables whose scopes are not restricted to a particular part of a program
Source Program Program in it's untranslated, original form
Object Program Translated version of a program
