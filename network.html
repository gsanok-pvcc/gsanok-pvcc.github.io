Networking and the Internet

Term Definition
Network Classifications
  • LAN:Local Access Network
  • MAN: Metropolitan Area Network
  • WAN: Wide Area Network
Network Models
  • Bus Network: Machines are all connected to a common communication line called a bus.
  • Star Network: One machine serves as a central focal point to which all the otheres are connected.
Protocols
  • CSMA/CD Ethernet: Protocol dictates each message be broadcast to all machines on the bus.
  • CSMA/CA WiFi: Wireless network avoids collisions rather than try to detect them.
Combining Networks
  • Repeater: Passes signals back and forth between two buses
  • Bridge: Morecomplex version of repeater; does not pass all messages across the connection
  • Switch: A bridge that connects several buses rather than just two
  • Router: Provide links between networks while allowing each to maintin it's own characteristics
  • Gateway: The "point"" at which one network is linked to an internet
Client/Server Model
  • P2P: peer-to-peer model has processes that provide service to and recieve service from each other
  • Distributed System: Consists of software units that execute as processes on different computers
  • Cloud Computing: Huge pools of shared computers can be allocated for clients as needed
Internet Adressing
  • Tier 1: ISP's that provide high-speed WANs
  • Tier 2: More regional ISPs with less scope and capability
  • Tier 3: An independent internet, called and intranet, that provides service to single homes or businesses
IP Adress
  • ICANN: ICANN home
  • DNS: Domain Name System; internet wide directory system
  • TLD: Top-Level Domain; one of highest domains in domain name system
  • Country-Code TLDs: World Standards
Internet Protocols
  • FTP:File Transfer Protocol
  • HTTP:Hyper Text Transfer Protocol
  • SMTP:Simple Mail Transfer Protocol
World Wide Web
  • URL: Uniform Resource Locator is each documents unique address
  • Client-Side Activities: Actions performed by a client through a browser
  • Server-Side Activities: Activities performed by a webserver
Internet Four-Layer Software
  • Application Layer: Software such as clients and servers that use internet to carry out tasks
  • Transport Layer: Used by the application layer to transfer send and recieve messages over the internet
  • Network Layer: Recieves packets from transport layer to send
  • Link Layer: Has the responsibility to transfer the packets from the network layer
Internet is:
  • Packet-Switching Network
  • Fault Tolerant
  • Redundant
Cybersecurity
  • Malware: Malicious software
  • Virus: Infects computer by inserting itself into existing programs
  • Worm: Autonomous program that transfers itself through network
  • Trojan Horse: Enters computer disguised as desireable program
  • Spyware: Collects information about the activities on a computer and reports back to instigator of attack
  • Phishing: Technique of obtaining information by asking for it or "casting lines"
  • DDoS Attack: Overloads a mass number of cumputers with messages at the same time
Protection
  • Firewall: Filters traffic passing through a point in the network
  • Spam Filter: Firewall designed to block unwanted emails
  • Proxy Server: Acts as an intermediary between a client and a server to shield client
  • Antivirus Software: Detects and removes the presence of known viruses and infections
  • Cryptography: Study of sending and receiving messages securely in the presence of adversaries
  • Public Key Encryption: Where the key can be known but the encoded message remains secure behind a private key
