All About Github

Q: What is a Version Control System?

  • A: Software tools that help software teams manage changes to source code over time.

Q: What is Git?

  • A: Git is the standard for version control systems and tracks changes in computer files.

Q: What is GitHub?

  • A: GitHub is a code hosting platform for version control and collaboration.

Q: What is the difference between Git and GitHub?

  • A: Git is the software used to edit code, while GitHub is a repository for Git code used to upload for collaboration.

Q: Who started GitHub and how was it started?

  • A: Chris Wanstrath and Tom Preston-Warner needed a program to share their Git work, so created GitHub out of necessity.

Q: What company owns GitHub now?

  • A: GitHub is now owned by Microsoft, and has been since 2018.

Q: How much does GitHub cost?

  • A: GitHub can be used for free, but there are also options for paid memberships for extra uses.

Q: What is the Octocat?

  • A: The Octocat is the mascot of GitHub, it is part cat and part octopus.

Git/Github Terms

  • Repository: Contains all of your projects files and each files revision history.
  • Commit: An individual change to a file or set of files.
  • Fork: A new repository that shares code and settings with the original repository.
  • Push: Pushing is how you transfer files from a local repository to a remote one.
  • Pull Requests: Allow you to tell others about changes you have pushed to a GitHub repository.
  • Workflows: A configurable automated process that will run one or multiple jobs.
  • Issues: Allow you to track your work on GitHub.
  • Raw Button: Allows you to view or copy the content of a file wihtout any styling.
  • Blame Button: Used to show when each line of code was modified, and who was the last to modify it.
