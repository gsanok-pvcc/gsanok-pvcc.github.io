Operating Systems Terminology

Term Definition
Operating System The software that controls the overall operation of a computer.
Job Queue A storage organization in which jobs are ordered firt-in, first-out fashion.
FIFO First-in first-out ordering system
Interactive Processing Allow a program being executed to carry on dialogue with the user through remote terminals
Time-Sharing Systems that provide service to multiple users at the same time.
Multiprogramming Divided time into intervals, with one job being set to one interval at a time.
Application Software The programs for performing tasks particular to the machine's utilization
System Software Performs the tasks that are common to computer systems in general.
Utility Software Software units that extend or customize the capabilities of the operating system.
Graphical User Interface (GUI) Objects to be manipulated, such as files or programs, are represented by icons on the display.
Shell User interface that communicates through textual messages.
Kernel The internal part of an operating system
Device Driver Software units that communicate with controllers to carry out operations on devices attached to the machine
File Manager Maintains records of all the files stored in mass storage, and where each file is located
Memory Manager Coordinates the machine's use of main memory
Virtual Memory Large "fictional" memory space created by paging
Boot Strapping Procedure that transfers the operating system from mass storage into main memory
ROM read-only-memory: its contents can be read but not altered
Deadlock Condition where two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
Spooling Technique of holding data for output at a later but more convenient time
Super User (Administrator) Had access to the operating system by identifting themselves as the administrator through username and password
Multi-Factor Authentication Needing more than one pice of authentication to access an account or device
